# Space Repetition

This project is a spaced repetition application built with Next.js. It helps users efficiently learn and memorize information through a card-based system.

## Features

- **Card Management**: Create, read, update, and delete cards.
- **Spaced Repetition Algorithm**: Utilizes an algorithm to optimize learning intervals.
- **Responsive Design**: Works seamlessly on various devices.

## Getting Started

### Prerequisites

- Node.js (version 14 or later)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd space-repetition
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   - Create a `.env.local` file in the root directory and add your configuration settings.

### Running the Application

To start the development server, run:
```
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.

### Building for Production

To build the application for production, run:
```
npm run build
```
